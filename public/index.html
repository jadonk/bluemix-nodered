Using Node-RED with the IBM Internet of Things Cloud QuickStart

Use Node-RED to build an application using the data from any device that you have connected to the IBM Internet of Things Cloud.


Sample application

The sample has a ready-made flow that processes temperature readings from a simulated device. The flow checks these readings against a threshold, then tells you whether the temperature is safe. Imagine that we're monitoring a CPU temperature from a real device.

To work through the sample application, follow the Node-RED IoT sample application recipe.


Use your own device data

Use the MAC address of any device that you have already connected to the IBM Internet of Things Cloud.There are recipes available for Raspberry Pi, ARM mbed, Intel Galileo and TI BeagleBone & SensorTag.

To view the complete list of IBM Internet of Things Cloud recipes go to the IBM IoT developerWorks site.
